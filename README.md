# VUE TS BoilerPlate

## Node - >= v16.15.1

## Build
```
yarn
yarn build-prod
```

## Build Preview
```
yarn build-prod || yarn build-dev
yarn preview
```

## Dev mode
```
yarn serve
```

### Kill 3000 and 8000 ports processes
```
yarn kill
```

### Lint
```
yarn lint | yarn lint --fix
```

### Tests
```
yarn test
```
